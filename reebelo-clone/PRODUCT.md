# Product

## Register

product

## Users
Mobile-first phone shoppers in Rwanda who want to compare Samsung Galaxy, iPhone, and Google Pixel devices quickly, then contact DODO TECHNOLOGIES through WhatsApp or Instagram.

## Product Purpose
DODO TECHNOLOGIES is a responsive phone storefront and lightweight admin tool. The public site should make it easy to browse phones, understand simple specs, see RWF pricing, and start a WhatsApp conversation with the selected product details. The admin panel lets staff add phones with Cloudinary-hosted images.

## Brand Personality
Confident, direct, and retail-ready. The brand uses a bold red DODO wordmark inspired by the supplied rounded type reference, balanced with clean teal commerce accents.

## Anti-references
Avoid generic marketplace clutter, tiny mobile product tiles, decorative animations that slow shopping, low-contrast text, and admin forms that feel cramped on phones.

## Design Principles
1. Mobile conversion first: primary actions must be obvious and thumb-friendly.
2. Show real product context: phone images, names, simple specs, and price must stay visible together.
3. Trust before flourish: clear warranty, condition, contact, and support cues matter more than decoration.
4. Staff speed matters: the admin upload flow should be simple enough to use from a phone.
5. Preserve the DODO identity: red logo, clean surfaces, and WhatsApp-forward actions should remain consistent.

## Accessibility & Inclusion
Target WCAG AA+. Maintain strong contrast, visible focus states, readable mobile type, 44px touch targets where practical, reduced-motion support, and layouts that do not require precision taps.
